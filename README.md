
# homebrew-tools

Homebrew formula for installing Astrokube's apps & tools.

## How To Use

Add repository:

```
brew tap astrokube/tools
```

Install [Kubebyuilder Initializer plugin](https://github.com/astrokube/kubebuilder-initializer-plugin):

```
brew install kubebuilder-initializer-plugin
```

Upgrade the Kubebuilder Initializer Plugin  to the latest version:

```
brew upgrade kubebuilder-initializer-plugin
```
